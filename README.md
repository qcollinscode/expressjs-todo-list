# ExpressJS Basic Todo List

### Live Demo: https://expressjs-basic-todo-list.herokuapp.com/

## Installation

Git clone this repository:

```
git clone https://github.com/qcollinscode/nodejs-expressjs-basic-todo-list.git
```

Change directory into the cloned repository:

```
cd nodejs-expressjs-basic-todo-list
```

Install the dependencies that are listed in the package.json file:

```
npm install
```

## Running

Start the application:

```
npm start
```
